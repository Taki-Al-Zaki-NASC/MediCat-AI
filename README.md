## 🛠️ Built With

- `Kotlin` – Cross-platform development  
- `Firebase & AWS` – Cloud, database, and authentication  
- `Python + GPT API` – Health-focused conversational AI  
- `TensorFlow Lite` – Future updates for offline AI
